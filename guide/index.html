                                            # FreeSql

                                            FreeSql是功能强大的 .NET ORM，支持 .NetFramework 4.0+、.NetCore 2.1+、Xamarin等支持 NetStandard 所有运行平台。

                                            支持 MySql/SqlServer/PostgreSQL/Oracle/Sqlite/Firebird/ClickHouse/达梦/南大通用GBase/神通/人大金仓/翰高/华为GaussDB/MsAccess 等数据库，以及自定义适配其它数据库。

                                            # 学习指南

                                            FreeSql 除了支持基本的增删查改功能外，还支持基于现有数据库创建模型（DbFirst），和支持基于模型创建数据库（CodeFirst)。
